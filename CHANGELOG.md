# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-22

### Added
- 20+ pre-built FAQ templates with unique designs
- AI-powered template generation via Anthropic Claude
- Glassmorphism admin panel with setup wizard
- Markdown FAQ answers with live preview editor
- Multi-user admin with role management (admin/viewer)
- JSON/CSV export and import with merge/replace modes
- Full-text search with SQLite FTS5 and relevance ranking
- SEO: meta tags, Open Graph, FAQ schema (JSON-LD), sitemap.xml, robots.txt
- Drag-and-drop sorting for products, categories, and FAQ items
- Draft/published/archived status for all content
- "Was this helpful?" feedback system with admin dashboard
- Custom CSS injection for template customization
- API rate limiting (public, auth, admin)
- RESTful API with OpenAPI 3.0 documentation and Swagger UI
- AI/Chatbot integration endpoints (POST /ask, GET /context)
- API key management with hashed storage
- Docker support with multi-stage build
- Interactive bash installer (`install.sh`)
- White-label branding with color customization
- Configurable text labels and header display options
- Product and category hierarchy with subcategories
